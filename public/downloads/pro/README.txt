YKS Pro downloads
=================

Put members-only files here: high-resolution PDFs, editable exports, and
Pro-exclusive items. Everything under /downloads/pro/ is gated at the CDN by
netlify/edge-functions/pro-gate.js — only logged-in Pro members can reach it.

Reference a file from a guide's frontmatter, e.g.:
  highResPdf: "/downloads/pro/daily-habit-tracker-hires.pdf"

See PRO.md for the full setup.
